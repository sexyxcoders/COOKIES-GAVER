## Installation

### Deploy on Heroku

1. Click the **Deploy to Heroku** button above.
2. Fill in required **Config Vars**:

| Key | Value |
|-----|-------|
| BOT_TOKEN | Your bot token |
| DATABASE_URI | Your database URI |
| REDIS_URI | redis://127.0.0.1:6379 (if needed) |
| NODE_ENV | development |

3. Click **Deploy App**.
